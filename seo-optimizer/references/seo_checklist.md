# 📈 SEO Checklist (2024/2025 Edition)

This checklist follows modern search engine guidelines and focuses on user-centric search performance.

## 1. Technical Health
- [ ] **HTTPS**: Ensure the site is served over SSL.
- [ ] **Mobile Responsiveness**: Test on multiple devices via Browser Tool.
- [ ] **Load Speed**: Optimize images (Next.js `<Image />`) and minimize main thread work.
- [ ] **Robots.txt**: Allow crawling of public pages, block private/admin areas.
- [ ] **Sitemap**: Clean, up-to-date, and submitted to search consoles.
- [ ] **Canonicalization**: Use `rel="canonical"` to point to the primary version of a URL.

## 2. On-Page Essentials
- [ ] **Title Tags**: Primary keyword + secondary + brand. Keep under 60 chars.
- [ ] **Meta Descriptions**: Active voice, value proposition, under 160 chars.
- [ ] **Heading Hierarchy**: H1 (Main Topic) -> H2 (Subtopic) -> H3 (Section). Never skip levels.
- [ ] **URL Slugs**: Lowercase, hyphen-separated, keyword-rich.
- [ ] **Internal Linking**: Minimum 3 internal links to other relevant content per page.

## 3. Media Optimization
- [ ] **Alt Attributes**: Descriptive, accurate, and keyword-integrated for ALL images.
- [ ] **File Size**: Images should be < 100KB where possible.
- [ ] **Responsive Images**: Use `srcset` or the Next.js `Image` component.

## 4. Content Quality (E-E-A-T)
- [ ] **Experience**: First-hand knowledge showcased in content.
- [ ] **Expertise**: Factually accurate, well-researched info.
- [ ] **Authoritativeness**: Clear author bios and source citations.
- [ ] **Trustworthiness**: Contact info, privacy policy, and secure checkout.

## 5. Schema.org (JSON-LD)
- [ ] **Product Schema**: Name, price, availability, and reviews.
- [ ] **FAQ Schema**: To gain rich snippets in search results.
- [ ] **Organization Schema**: Logo, social links, and contact info.
- [ ] **Article Schema**: For blog posts and research articles.

## 6. Local SEO (if applicable)
- [ ] **Google Business Profile**: Sync info with website.
- [ ] **NAP**: Name, Address, Phone number consistency.
